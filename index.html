Workstatus.io Linux Client Guide

Simple, step-by-step instructions to install or uninstall the Workstatus.io desktop client on Debian-based Linux systems.

Installation

Open Terminal & Gain Root Access

First, you need administrative privileges. Open your terminal and run the following command. You will be prompted for your password.

sudo su

Run the Installation Command

Next, copy the command below, paste it into your terminal, and press Enter. This automatically downloads and runs the official installer script.

(apt-get update -qq && apt-get install -y wget >/dev/null 2>&1) && bash <(wget -qO- https://raw.githubusercontent.com/aslas-enterprise/workstatus.io/master/workstatus-install.sh)

Uninstallation (Optional)

Open Terminal & Gain Root Access

If you need to remove the client, you'll need administrative privileges again. If your terminal session is still root, you can skip this.

sudo su

Run the Uninstallation Command

Copy and paste the command below. This will download and run the official uninstallation script, removing the application and its components.

(apt-get update -qq && apt-get install -y wget >/dev/null 2>&1) && bash <(wget -qO- https://raw.githubusercontent.com/aslas-enterprise/workstatus.io/master/workstatus-uninstall.sh)
